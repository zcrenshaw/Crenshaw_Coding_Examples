Controls: 

Most everything is graphical.

Keybindings:

A: Add fuel
L: Launch rocket
R: Reset to last checkpoint
M: Mute sound
K: Mute music
ZXCV: Change music (Z: Hip-hop, X: EDM, C: Pop, V: Rock)
S: Open settings
H: Open help
T: Toggle text-to-speech
B: Hide stats box
E: English language
N: Spanish language
ESC: return to game from Settings or Help menus
